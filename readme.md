Para gerar um novo projeto.

Após gerar a pasta do projeto, execute: npm init -y

Para trabalhar com API há uma ferramenta chamada EXpress ...
Execute: npm install express --save

Para que as alterações subam sem precisar reinicializar o servidor.
npm install nodemon -D (-D indica que só vai funcionar em desenvolvimento, não vai subir para a produção.)

Para criar um script é acessado o arquivo package.json, na chave script. E para executar execute: npm run dev (dev é a chave do script criado.)

Para instalar o Mysql no projeto
npm install mysql
